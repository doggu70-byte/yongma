## 시작하기

### 환경 설정

```bash
# 의존성 설치
pip install -r requirements.txt

# 데이터베이스 마이그레이션
python manage.py migrate

# 개발 서버 실행 (포트 0804)
python manage.py runserver 0:0804
```

### 개발 서버 접속
- **로컬**: http://localhost:0804
- **네트워크**: http://[IP주소]:0804
